Buttons

Buttons are a very common component in building user interactions. The button object is very flexible and extendable mixin that can easily be customized globally by changing the button variables within the settings.scss partial, or by providing optional over-ride arguments when calling the mixin on the fly.

Button Structure

/**
 * Button Structure
 *
 * Extends button object
 *
 * 6. Normalise box model styles.
 * 7. If the buttons text is 1em, and the button is (3 * font-size) tall, then
 *    there is 1em of space above and below that text. We therefore apply 1em
 *    of space to the left and right, as padding, to keep consistent spacing.
 * 8. Basic cosmetics for default buttons. Change or override at will.
 */

/**
 * Small Button 
 */
.btn--#{$small} {
  font-size: ($base-font-size - 4px);
  padding: 2px 6px; //6
  line-height: $base-line-height;//7
  margin: 0;//6
  border-radius: 3px;//8
}

/**
 * Medium Button 
 */
.btn--#{$medium} {
  font-size: ($base-font-size - 2px);
  padding: 5px 9px; //6
  line-height: $base-line-height;//7
  margin: 5px;//6
  border-radius: 5px;//8
}

/**
 * Large Button 
 */
.btn--#{$large} {
  font-size: ($base-font-size + 2px);
  padding: 9px 14px; //6
  line-height: $base-line-height;//7
  margin: 5px;//6
  border-radius: 7px;//8
}

/**
 * Full Button
 *
 * full screen, block level button style for small screens 
 */
.btn--#{$touch} {
  @extend .btn--#{large};
  display: block;
  width: 100%;
  padding-left: 0;
  padding-right: 0;
}

Button Skin

/**
 * Button Skin
 *
 * basic skin styles for prototyping
 * override these with your project specific design
 */
@mixin button-skin($button-skin) {
  @if $button-skin == #{$default} {
    background-color: $btn__default--color1;
    color: $white;
    border: 1px solid $strong;
    &:hover {
    background-color: $btn__default--color2;
    }
    &:active,
    &.active {
    background-color: tint($btn_default--color2, 10%);
    }
    &.disabled,
    &[disabled] {
    opacity: 0.6;
    cursor: default;
    }
  }
  @if $button-skin == #{$positive} {
    background-color: $btn__positive--color1;
    color: $white;
    border: 1px solid $strong;
    &:hover {
    background-color: $btn__positive--color2; 
    }
    &:active,
    &.active {
    background-color: tint($btn__positive--color2, 10%);
    }
    &.disabled,
    &[disabled] {
    opacity: 0.6;
    cursor: default;
    }
  }
 @if $button-skin == #{$negative} {
    background-color: $btn__negative--color1;
    color: $white;
    border: 1px solid $strong;
    &:hover {
    background-color: $btn__negative--color2; 
    }
    &:active,
    &.active {
    background-color: tint($btn__negative--color2, 10%);
    }
    &.disabled,
    &[disabled] {
    opacity: 0.6;
    cursor: default;
    }
  }
 @if $button-skin == #{$secondary} {
    background-color: $btn__secondary--color1;
    color: $white;
    border: 1px solid $strong;
    &:hover {
    background-color: $btn__secondary--color2; 
    }
    &:active,
    &.active {
    background-color: tint($btn__secondary--color2, 10%);
    }
    &.disabled,
    &[disabled] {
    opacity: 0.6;
    cursor: default;
    }
  }
}
