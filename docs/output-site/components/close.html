Close

The close component visually indicates an option to dismiss an alert, message, or modal within a site or app.

Close styles extend the close-object. See alerts for a common use case.

Markup Pattern

<span class="close close--small close--light"> ✕ </span>

Variables

/**
 * Close Variables
 * 
 * 1. optionally rename close skin styles
 */

/* Close Structure Namespace: */
$small-close:                         small;    /* 1 */
$large-close:                         large;    /* 1 */

/* Close Skin Namespace: */
$positive-close:                      light;    /* 1 */
$negative-close:                      dark;     /* 1 */

Close Structure

/**
 * Close Structure
 *
 * Extends close object
 */
.close--#{$small-close} {
  font-size: ms(1); 
  padding: $spacer-unit--quarter;
  right: 2px; 
  top: 2px;  
}

.close--#{$large-close} {
  font-size: ms(2); 
  padding: $spacer-unit--half;
  right: 4px; 
  top: 4px;  
}

Close Skin

/**
 * Close Skin
 */
.close--#{$positive-close} {
  color: $white;
  font-weight: bold;
  text-decoration: none;
  opacity: .2;
  &:hover,
  &:focus {
    opacity: .8;
    text-decoration: none;
  }
}

.close--#{$negative-close} {
  color: $black;
  font-weight: bold;
  text-decoration: none;
  opacity: .2;
  &:hover,
  &:focus {
    opacity: .8;
    text-decoration: none;
  }
}
