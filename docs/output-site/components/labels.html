Labels

Labels convey contextual emphasis and are an extension of the label-object

The .label class ... <strong> or <i> tags when you need to communicate important actions. Labels are easily used inline by applying the label class to the <span> element.

Markup Pattern

<span class="label label--inline label--info">label text</span>

Label Variables

/**
 * Label Variables
 * 
 * 1. optionally rename label skin styles
 */

/* label Structure Namespace: */
$small-label:                         inline;     /* 1 */
$large-label:                         block;      /* 1 */

/* label Skin Namespace: */
$default-label:                       info;       /* 1 */
$positive-label:                      success;    /* 1 */
$negative-label:                      error;      /* 1 */
$warning-label:                       warning;    /* 1 */

Label Structure

/**
 * Label Structure
 */
.label--#{$small-label} {
  line-height: $base-line-height;  
  display: inline;
  border-radius: 0px;
  padding: convert-to-em(1px) convert-to-em(4px) convert-to-em(2px);
  font-size: $base-font-size;
  font-weight: bold; 
}

.label--#{$large-label} {
  line-height: $base-line-height;  
  display: inline;
  border-radius: 0px;
  padding: convert-to-em(1px) convert-to-em(4px) convert-to-em(2px);
  font-size: $base-font-size; 
  font-weight: bold; 
}
