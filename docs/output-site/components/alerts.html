Alerts

Alert styles extend the alert object. Alerts are most commonly injected dynamiclly to provide context specific feedback to a specific user action. Alerts also make use of the close-object to allow users to dismiss the alert message. 

<div class="alert alert__small alert--default">This is a small default alert.<a href="#" class="close">&times;</a></div>

Alert Variables

/**
 * Alert Variables
 *
 * Optionally rename alert skin and structure classes
 */

/**
 * Alert Structure Namespace:
 */
$small:                               small;
$large:                               large;

/**
 * Alert Skin Namespace:
 */
$default:                             info;
$positive:                            success;
$negative:                            error;
$warning:                             warning;

Alert Structure

/**
 * Alert Structure
 *
 * Extends alert object
 */
.alert_#{$small} {
  padding: $spacer-unit--quarter;
  font-weight: bold;
  font-size: $base-font-size;
  margin: $spacer-unit--half 0;
  border-radius: 4px;
}

.alert__#{$large} {
  padding: $spacer-unit--half;
  font-weight: bold;
  font-size: $base-font-size - 2;
  margin: $spacer-unit--half 0;
  border-radius: 2px;
}
